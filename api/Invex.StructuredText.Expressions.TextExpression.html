Class TextExpression

Namespace
Invex.StructuredText.Expressions
Assembly
Invex.StructuredText.dll

The base type of the platform-agnostic expression tree. Expressions are immutable records composed via fluent methods and operators, then rendered to platform-specific text by a TextExpressionFormatter.

[PublicAPI]

Remarks

Construction is ergonomic thanks to implicit conversions from string, bool, and numeric types, plus fluent builders for functions (Contains(string), …) and logic operators (EqualTo(TextExpression), &, |, !, …).
