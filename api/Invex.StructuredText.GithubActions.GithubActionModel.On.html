A workflow trigger event (on). This is a discriminated union over the GitHub webhook events; construct the variant for the event you want, e.g. new On.Push { ... } or new On.Schedule(["0 6 * * 1"]). Enum member names intentionally match the lowercase/snake_case activity-type strings used in workflow YAML.
