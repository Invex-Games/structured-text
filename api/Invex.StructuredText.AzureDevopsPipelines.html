Namespace Invex.StructuredText.AzureDevopsPipelines

Classes

DevopsExpressionFormatter

Renders TextExpression trees using Azure DevOps expression syntax: logic and comparison operators render as functions (and(...), or(...), not(...), eq(...), ne(...), lt(...), …), EvaluateExpression becomes a compile-time template expression ${{ ... }}, DevopsRuntimeExpression becomes a runtime expression $[ ... ], and DevopsMacroExpression becomes a macro expansion $(variable). Step and job references render against the steps. and dependencies. contexts.

DevopsMacroExpression

Wraps a variable name to be expanded using Azure DevOps macro syntax: $(variableName)

DevopsPipelineWriter

Serializes a DevopsPipeline model to Azure Pipelines YAML. Expressions embedded in the model are rendered with DevopsExpressionFormatter.

DevopsRuntimeExpression

Wraps an expression to be evaluated at runtime using Azure DevOps runtime expression syntax: $[ expression ]

Invex.StructuredText: Strongly-typed YAML generation for GitHub Actions, Dependabot, and Azure DevOps Pipelines.
